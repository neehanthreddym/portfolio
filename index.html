AI-Powered Inventory Management System

Context: Personal project · FastAPI, PostgreSQL, MongoDB, LangChain, JWT · Hybrid DB architecture

Problem: Multi-warehouse inventory required both relational consistency (transactions, FIFO) and flexible document storage (audit logs, metadata), plus natural-language commands for operations.

Process: I built the FastAPI microservice and defined the hybrid schema: PostgreSQL for transactional inventory and MongoDB for logs. I implemented JWT auth, FIFO stock deduction, and audit logging. I developed a LangChain-based agent with tool-level permissions to interpret natural language and execute inventory operations safely.

Outcome: Single API for multi-warehouse inventory with JWT authentication, FIFO deduction, and audit trails; AI agent for natural-language inventory commands with permission controls.
