Featured System

Sentinel - AI News Intelligence In Progress

An automated, AI-driven news aggregation and synthesis platform.

Overview

Sentinel tackles information overload by automatically synthesizing news from diverse sources into cohesive daily digests. Its stateful, multi-agent LLM workflow guarantees that summaries are accurate, structured, and unbiased.

System Architecture & Orchestration

A cyclic, two-agent LangGraph workflow automates generation and quality control. The Researcher Agent drafts digests with strict JSON outputs, while the Editor Agent acts as an automated guardrail, reviewing drafts for hallucinations and routing them for revision.

Engineering & Data Pipeline

A robust pipeline scrapes clean HTML (NewsAPI, Trafilatura) with strong Pydantic validation, stored in a PostgreSQL database. Ultra-low latency inference is powered by the Groq API, with a custom event logger for full system observability.
