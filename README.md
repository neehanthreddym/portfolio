# Neehanth Reddy — AI & ML Engineer Portfolio (v2.0)

Welcome to the **v2 redesign** of my personal portfolio. This space is dedicated to showcasing my work as an AI Engineer specializing in **RAG pipelines, LLM agents, scalable ML systems, and robust Data Science backends**.

## 🚀 The v2 Redesign

This iteration marks a significant visual and structural departure from v1, transitioning to a high-contrast, developer-focused **"Space Black" aesthetic**. The goal was to create an interface that feels like a premium technical control plane—putting the architecture, systems, and data flow front and center.

### Key Technical Improvements:
* **DRY Component Architecture:** Completely refactored the CSS to utilize base `.card` and `.btn` utility classes, standardizing the UI while drastically cutting down visual regression risks.
* **Fluid Layouts:** Implemented explicit 4-column connected CSS Grids for technical capabilities, paired with wide `1536px` maximum desktop container widths.
* **Zero Framework Overhead:** Built entirely using semantic HTML5, Vanilla JavaScript, and raw CSS3 Custom Properties. No heavy React/Next.js overhead for a simple static presentation.
* **Smooth UX:** Integrated [Lenis](https://github.com/darkroomengineering/lenis) for performant, frictionless scrolling across anchor links.

## 🛠 Tech Stack

* **Front-end Structure:** Vanilla HTML5 
* **Styling Engine:** Raw CSS3 (Flexbox, Grid, Custom Properties, Media Queries)
* **Interactivity:** Vanilla JavaScript (`IntersectionObserver`, DOM Manipulation)
* **Typography:** `Inter` (UI) and `JetBrains Mono` (Technical details)
* **Assets:** FontAwesome and Devicon
